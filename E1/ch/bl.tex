	\begin{Citemize}
		\item Si estoy ingresando al sistema por medio de la aplicación móvil, ingresaré mi nombre de usuario y mi contraseña como se muestra en la pantalla \getElementById[UI]{UIMU01M}.
		\item Si estoy ingresando al sistema por medio de la aplicación web, ingresaré mi nombre de usuario y contraseña como se muestra en la pantalla \getElementById[UI]{UIMU01W}
	\end{Citemize}
	
	El sistema debe validar que:
		\begin{Citemize}
			\item Mi cuenta no este bloqueada como lo especifica la regla de negocio \getElementById{BR}{BR-MU001}. Si esta regla no se cumple el sistema me debe mostrar el mensaje \getElementById[MSG]{MSG01} en pantalla .
			
			\item No tenga una sesión activa en otro dispositivo como lo especifica la regla de negocio \getElementById{BR}{BR-MU002}. Si esta regla no se cumple el sistema me debe mostrar el mensaje \getElementById[MSG]{MSG02}en pantalla.
		\end{Citemize}
		
	El sistema debe determinar mi rol y redirigirme a la pantalla correspondiente como se indica a continuación:
		\begin{Citemize}
			\item Si estoy ingresando al sistema por medio de la aplicación móvil como \textit{Cliente}, el sistema debe determinar mi posición a través del GPS, obtener una lista de al menos una cafeteria y máximo 5 que este cerca de mi posición en un radio de 3 kilómetros y mostrármelas como se muestra en la pantalla \getElementById[UI]{UIMU02M}. Si no se encuentra almenos una cafetería la aplicación móvil me debe mostrar el mensaje \getElementById[MSG]{MSG03} en pantalla.
		\end{Citemize}
